HOW TO PLAY GAME

  • To play a maze game, start by understanding the objective, usually navigating from a starting point to the exit.

  • User have to reach the destination in limited period of time.

  • ⬆️ :Upper Arrow Key (Move Forward)

  • ⬇️ : Down Arrow Key (Move Downwards)

  • ⬅️ / ➡️: Left Arrow Key / Right Arrow Key (To Move Sideways)
